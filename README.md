# Kenyan County Health Dashboard 2026

Interactive visualization of key public health indicators across major Kenyan counties. Built with React, TypeScript, and Recharts.

## Features
- Real-time filtering of health metrics by county
- Interactive Bar Charts for comparative analysis
- Key Performance Indicators (KPIs) for Vaccination, Maternal Mortality, and Malaria
- Responsive mobile-first design using Tailwind CSS

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the application: `npm start`

## Data Points
- **Vaccination Rate**: Percentage of children fully immunized.
- **Maternal Mortality**: Deaths per 100,000 live births.
- **Malaria Incidence**: Reported cases per 1,000 residents.
